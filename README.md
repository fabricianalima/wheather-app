## ✨ Funcionalidades

✅ **Busca por Cidade** - Digite o nome de qualquer cidade do mundo  
✅ **Dados em Tempo Real** - Informações atualizadas via API Open-Meteo  
✅ **Temperatura** - Exibe a temperatura atual em graus Celsius  
✅ **Umidade** - Mostra o percentual de umidade do ar  
✅ **Velocidade do Vento** - Velocidade em km/h  
✅ **País** - Exibe o país da cidade pesquisada  
✅ **Interface Responsiva** - Funciona perfeitamente em celulares e tablets  
✅ **Suporte a ENTER** - Pressione ENTER para buscar além do botão  
✅ **Tratamento de Erros** - Mensagens amigáveis quando algo der errado  
✅ **Design Moderno** - Interface com gradiente e animações suaves  

---

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura semântica
- **CSS3** - Estilos responsivos com gradientes e animações
- **JavaScript (ES6+)** - Lógica da aplicação com async/await
- **API Open-Meteo** - Dados meteorológicos gratuitos
